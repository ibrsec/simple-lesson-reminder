<a name="quick-setup"></a>
## 🛫 Quick Setup

```sh
# clone the project
git clone https://github.com/ibrsec/lesson-reminder.git

# enter the project directory
cd lesson-reminder

# install dependency
npm install || yarn install

# develop
npm run dev || yarn start
```
